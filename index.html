광고 전체 상황 — 채널별 비용·매출
전체 이슈와 광고가 어떻게 연결되는지 파악

광고 지표를 볼 때 — 볼륨 vs 효율

📦 볼륨 (얼마나 했나)
💸 비용
👁️ 노출수
🖱️ 클릭수
🛒 구매수
💰 매출액
⚡ 효율 (얼마나 잘 했나)
📡 CPM
🖱️ CPC
📊 CTR
📈 ROAS
🔄 CVR

볼륨만 보면 "많이 썼다"는 건 알지만 "잘 썼는지"를 모름.
효율만 보면 "효율은 좋은데 매출이 작은" 경우를 놓침. 두 가지를 같이 봐야 함.

채널 주로 보는 지표

네이버 검색광고 비용, 클릭수, 전환수, 전환매출, ROAS, CPC
GFA 비용, 노출, 클릭, 전환
카카오 비용, 전환
구글 비용, 전환, 매출

이슈의 원인을 찾고, 어떤 액션이 어떤 결과를 만들었는지 정리

기간 흐름에 따라 지표 정리 (선그래프 방식)

📉
아래 지표들을 날짜 축으로 시각화
단순히 숫자 나열이 아니라, 시간 흐름에 따른 변화를 선으로 이어서
어느 시점에 무슨 일이 있었는지 바로 보이게 만든다.
💸 비용 📡 CPM 🖱️ 클릭수 💵 CPC 📊 CTR 🛒 구매수 💰 매출 📈 ROAS

기간별 액션 정리

🧪
테스트 액션

새 소재, 새 캠페인, 새 타겟팅 등을 시험했던 기간과 결과 정리.

⚙️
운영 액션

예산 증감, 입찰가 조정, 비효율 광고 off 등 운영 최적화 내역.

🎨
소재 액션

새 소재 등록, 기존 소재 교체, A/B 테스트 등 크리에이티브 변화.

💡

→ 향후 [대응 방향]으로 접근 예정."
4
향후 액션 — 성과를 높이는 방향으로
과거 데이터 기반 · 성과 개선이 목적 · 우선순위 명확하게

액션 우선순위 판단 기준

HIGH
지금 성과를 직접 좌우하는 것

ROAS가 낮은 채널 예산 조정, 전환율이 급락한 소재 교체, 비용 대비 효율이 가장 나쁜 캠페인 off

MID
성과에 간접적으로 영향 주는 것

타겟 확장 테스트, 새 소재 A/B 테스트, 랜딩페이지 개선 검토

LOW
중장기 방향을 위한 것

신규 채널 검토, 브랜딩 소재 기획, 추적 구조 개선

액션 작성 원칙

📊
데이터 근거 있어야 함

"느낌"이 아니라 실제 지표 변화를 근거로 액션을 제안. "ROAS가 3주 연속 하락했으니 소재 교체" 형태로.

🎯
성과가 높아지는 방향

매출 증가, ROAS 개선, 마진 확대 중 어느 목표를 향한 액션인지 명시.

📅
실행 기간 포함

언제 실행하고, 언제 결과를 볼 것인지 기간 설정. "이번 주 내", "다음 2주" 등.

🗺️ 전체 흐름 한눈에

1️⃣
전체 상황 → 매출·비용·ROAS·마진 단기/중기/장기로 이슈 파악
2️⃣
광고 전체 → 이슈와 연결되는 채널별 비중 확인, 집중 채널 선정
3️⃣
세부 채널 → 볼륨+효율 지표 시계열로 보고, 액션과 결과 매핑, 요약 작성
4️⃣
향후 액션 → 데이터 기반, 성과 개선 방향, 우선순위 붙여서 정리
